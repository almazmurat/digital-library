# Test Case: TC004

**Title:** Verify Session Timeout

**Objective:** Ensure that user sessions expire after a period of inactivity.

**Preconditions:**

- User is logged in.

**Test Data:**

- Session timeout duration: 15 minutes

**Steps:**

1. Log in with valid credentials.
2. Remain inactive for 15 minutes.
3. Attempt to perform an action.

**Expected Result:**

- User is logged out automatically.
- User is redirected to the login page.

**Priority:** Medium

**Linked Risk IDs:** R9
