# Test Case: TC002

**Title:** Verify Password Reset

**Objective:** Ensure that users can reset their password successfully.

**Preconditions:**

- User account exists in the database.
- User has access to the registered email.

**Test Data:**

- Email: valid_user@example.com

**Steps:**

1. Navigate to the password reset page.
2. Enter the registered email address.
3. Submit the password reset request.
4. Check the email inbox for the reset link.
5. Click the reset link and set a new password.

**Expected Result:**

- Password reset email is sent.
- User can set a new password and log in successfully.

**Priority:** High

**Linked Risk IDs:** R1
