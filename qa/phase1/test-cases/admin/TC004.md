# Test Case: TC004

**Title:** Verify admin integration failure is logged and surfaced safely

**Objective:** Ensure external integration failures do not expose sensitive errors and are logged.

**Preconditions:**

- Admin is logged in.
- Integration endpoint is unavailable or mocked to fail.

**Test Data:**

- Integration sync action in admin panel.

**Steps:**

1. Trigger integration action from admin page.
2. Observe UI response.
3. Review application log for failure entry.

**Expected Result:**

- UI shows controlled error message.
- Failure is logged with actionable context.
- No stack trace or secrets are exposed in UI.

**Priority:** High

**Linked Risk IDs:** R6, R15
