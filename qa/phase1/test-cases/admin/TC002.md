# Test Case: TC002

**Title:** Verify admin access is blocked for non-admin user

**Objective:** Ensure non-admin roles cannot access admin management pages.

**Preconditions:**

- Reader or librarian account exists.
- User is logged in as non-admin.

**Test Data:**

- URL: /admin

**Steps:**

1. Sign in as a non-admin user.
2. Navigate to /admin.
3. Observe response and destination.

**Expected Result:**

- Access is denied (redirect or forbidden response per route behavior).
- Admin dashboard content is not shown.

**Priority:** High

**Linked Risk IDs:** R14, R4
