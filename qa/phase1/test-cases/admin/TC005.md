# Test Case: TC005

**Title:** Verify admin session timeout requires re-authentication

**Objective:** Ensure expired admin session cannot execute management actions.

**Preconditions:**

- Admin account exists and can log in.

**Test Data:**

- Admin session timeout window from config.

**Steps:**

1. Log in as admin and open admin page.
2. Wait until session expiration.
3. Attempt to publish or update content.

**Expected Result:**

- Action is blocked.
- User is redirected to login or receives unauthorized response.
- No admin change is persisted after expiration.

**Priority:** High

**Linked Risk IDs:** R1, R9, R14
