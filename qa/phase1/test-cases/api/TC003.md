# Test Case: TC003

**Title:** Verify API Rate Limiting

**Objective:** Ensure that API rate limiting is enforced.

**Preconditions:**

- Rate limiting policy is enabled.

**Test Data:**

- Endpoint: /api/v1/resource
- Rate Limit: 100 requests per minute

**Steps:**

1. Send 101 requests to the endpoint within one minute.

**Expected Result:**

- The 101st request returns 429 Too Many Requests.
- Retry-After header is included in the response.

**Priority:** High

**Linked Risk IDs:** R4
