# Test Case: TC002

**Title:** Verify API Authentication

**Objective:** Ensure that API endpoints require authentication.

**Preconditions:**

- API key is available.

**Test Data:**

- Endpoint: /api/v1/resource
- API Key: InvalidKey123

**Steps:**

1. Send a request to the endpoint without an API key.
2. Send a request with an invalid API key.

**Expected Result:**

- Requests without a key return 401 Unauthorized.
- Requests with an invalid key return 403 Forbidden.

**Priority:** High

**Linked Risk IDs:** R4
