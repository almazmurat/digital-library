# Test Case: TC005

**Title:** Verify API denies request without session authentication

**Objective:** Ensure protected account API endpoints reject unauthenticated access.

**Preconditions:**

- No active authenticated session.

**Test Data:**

- Endpoint: /api/v1/account/reservations

**Steps:**

1. Send a GET request to the endpoint without login/session cookies.
2. Record response status and payload.

**Expected Result:**

- Response status is 401 Unauthorized.
- Response body indicates authentication is required.

**Priority:** High

**Linked Risk IDs:** R4, R14
