# Test Case: TC004

**Title:** Verify API Response Time

**Objective:** Ensure that API response times meet performance requirements.

**Preconditions:**

- API is running.

**Test Data:**

- Endpoint: /api/v1/resource
- Expected Response Time: < 200ms

**Steps:**

1. Send a request to the endpoint.
2. Measure the response time.

**Expected Result:**

- Response time is less than 200ms.

**Priority:** Medium

**Linked Risk IDs:** R4
