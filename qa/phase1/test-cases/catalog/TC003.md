# Test Case: TC003

**Title:** Verify Search with No Results

**Objective:** Ensure that the catalog search handles queries with no matching results gracefully.

**Preconditions:**

- Catalog contains indexed items.
- User is logged in.

**Test Data:**

- Search Query: "NonExistentBook"

**Steps:**

1. Enter a query that does not match any catalog items.
2. Click the search button.

**Expected Result:**

- No results are displayed.
- A message indicating no results is shown.

**Priority:** Medium

**Linked Risk IDs:** R2
