# Test Case: TC005

**Title:** Verify Search with Special Characters

**Objective:** Ensure that the catalog search handles special characters in queries correctly.

**Preconditions:**

- Catalog contains indexed items.
- User is logged in.

**Test Data:**

- Search Query: "@#$%^&\*()"

**Steps:**

1. Enter a query with special characters.
2. Click the search button.

**Expected Result:**

- Search results are displayed without errors.
- Results match the query if applicable.

**Priority:** Medium

**Linked Risk IDs:** R2
