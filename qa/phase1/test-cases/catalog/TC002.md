# Test Case: TC002

**Title:** Verify Advanced Search Filters

**Objective:** Ensure that advanced search filters work as expected.

**Preconditions:**

- Catalog contains indexed items with various attributes.
- User is logged in.

**Test Data:**

- Filter: Author = "John Doe"

**Steps:**

1. Navigate to the advanced search page.
2. Select the author filter and enter "John Doe."
3. Click the search button.

**Expected Result:**

- Search results are displayed.
- Results are filtered by the selected author.

**Priority:** High

**Linked Risk IDs:** R2
