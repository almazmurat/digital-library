# Test Case: TC004

**Title:** Verify Search Pagination

**Objective:** Ensure that catalog search results are paginated correctly.

**Preconditions:**

- Catalog contains more than 50 items.
- User is logged in.

**Test Data:**

- Search Query: "Science"

**Steps:**

1. Enter a query that returns more than 50 results.
2. Click the search button.
3. Navigate to the second page of results.

**Expected Result:**

- Results are paginated.
- The second page displays the next set of results.

**Priority:** Medium

**Linked Risk IDs:** R2
