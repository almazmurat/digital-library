# Metrics

Quantitative tracking for research paper.

## Files
- aseline-metrics.csv — Week-by-week metric snapshots (see column definitions inside)
- charts/ — Generated charts (PNG/SVG) for research paper

## Key Metrics Tracked (M1–M12)
1. Test case count (by type)
2. PHPUnit line coverage %
3. Mutation score (MSI)
4. CI pass rate %
5. P0 defects found/fixed
6. API P50/P95 response time
7. E2E test execution time
8. PHPUnit suite execution time
9. Risk coverage %
10. Automation ratio %
11. False positive rate (flaky tests)
12. Defect detection efficiency
