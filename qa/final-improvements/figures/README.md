# Final Improvements Rendered Figures

This directory contains rendered PNG outputs generated from the current source files in `qa/final-improvements/figure-sources/`.

## Purpose

- Provide paper-ready raster image assets for direct insertion.
- Keep final-improvements outputs isolated from legacy phase figure folders.
- Maintain deterministic source-to-output mapping.

## Source and Output Convention

- Source: `qa/final-improvements/figure-sources/figure-XX-*.md`
- Output: `qa/final-improvements/figures/figure-XX-*.png`

## Render Audit

See `qa/final-improvements/figures/rendering-report.md` for:

- commands used,
- per-figure render status,
- compatibility fixes,
- insertion-readiness notes.
