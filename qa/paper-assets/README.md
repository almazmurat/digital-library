# QA Paper Assets

This folder contains publication-ready assets for the final research paper.

## Structure

- `figures/phase1/` — baseline QA layer (Phase 1) charts
- `figures/phase2/` — automation and CI governance layer (Phase 2) charts
- `figures/midterm/` — Intermediate Empirical Review charts
- `figures/phase3/` — experimental evaluation layer (Phase 3) charts
- `figures/summary/` — cross-phase summary charts

## Generation

Figures are generated from factual CSV datasets using:

`python qa/paper-assets/generate-paper-figures.py`

## Index

See `qa/paper-assets/figures/figure-index.md` for figure-to-dataset mapping and paper section usage.
